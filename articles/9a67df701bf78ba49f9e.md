## はじめに
Vimでファイルの比較をしたい。

## コマンド
```
vimdiff file1 file2 
```

```
vim -d file1 file2
```

### 差分に移動する
[c 次の差分に移動する。
]c 前の差分に移動する。

### 差分の反映
dp 自身の内容をもう一方のファイルに反映する。
do もう一方のファイルに自身のファイル内容を反映する。
